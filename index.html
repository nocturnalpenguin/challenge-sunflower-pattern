Sunflower pattern

In order to describe the pattern of seeds in a sunflower head, german physicist Helmut Vogel proposed a formula

φ = n * 137.5
r = c * √n

where:

  • n is the ordering number of a seed, counting outward from the center
  • φ is the angle between a reference direction and the position vector of the nth seed in a polar coordinate system originating at the center
  • 137.5 is a divergence angle between the position vectors of any two successive seed.
    this angle is also known as "Golden Angle"
  • r is the distance between the center and the nth seed,
    given constant scaling parameter c
